#  Banco de Dados – TechZada Lan House & Suporte Full-Stack

Este repositório faz parte da disciplina de Banco de Dados e contém o projeto SQL desenvolvido para uma empresa fictícia chamada **TechZada**, uma lan house moderna que oferece diversos serviços: acesso a computadores, internet, impressão, xerox, suporte técnico e até ajuda especializada com programadores full-stack.

---

##  Objetivo do Projeto

O objetivo deste trabalho foi:
- Criar um banco de dados realista usando comandos **DDL** e **DML**.
- Organizar um projeto completo no GitHub, com estrutura profissional.
- Entender como funcionam tabelas, chaves primárias, estrangeiras e inserção de dados.
- Aplicar conceitos práticos aprendidos em aula.

Durante o desenvolvimento, foi possível aprender como:
- Criar bancos e tabelas no MySQL.
- Manipular dados usando `INSERT`, `UPDATE`, `DELETE` e `SELECT`.
- Utilizar relacionamentos entre tabelas (clientes, serviços e atendimentos).
- Organizar arquivos SQL dentro de um repositório.

##  Como Executar o Script SQL

1. Abra seu gerenciador MySQL (MySQL Workbench, phpMyAdmin ou XAMPP).
2. Importe ou copie o conteúdo do arquivo: sql_scripts/techzada_mysql.sql

3. Execute o script.
4. O banco **techzada** será criado automaticamente.
5. As tabelas `clientes`, `servicos` e `atendimentos` irão aparecer.
6. Os dados iniciais serão inseridos automaticamente.

---

##  Sobre o Banco de Dados

O banco é composto por três tabelas principais:

### **1. clientes**
Guarda informações sobre os clientes da lan house.

### **2. servicos**
Lista todos os serviços oferecidos pela empresa (impressão, acesso ao computador, suporte técnico, etc.).

### **3. atendimentos**
Registro dos atendimentos realizados, incluindo:
- cliente atendido  
- serviço prestado  
- data  
- funcionário responsável  

Este modelo permite consultas úteis, como:
- histórico de serviços por cliente  
- quais serviços são mais utilizados  
- controle interno de atendimentos
